# 英文学习助手 ELA

English learner Assistant

- chrome extentions
- base on OpenAI TTS (text-to-speech) model [https://platform.openai.com/docs/guides/text-to-speech]

## 主要功能

- 将选择中的文字读出来（OpenAI TTS）
