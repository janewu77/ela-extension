## 详细描述

```
外语文章读起来很吃力？ELA 用 AI（GPT-4o）帮你翻译、查词、朗读——一切都在简洁的侧边栏里完成。支持英语、德语、法语、日语、西班牙语等任何你正在学习的语言。自定义 AI 提示词，打造专属学习体验。

只需选中任意文字，侧边栏即刻捕获内容，随时可以收听、翻译或查词。无论你是在啃英语词汇的学习者、阅读德语新闻的语言爱好者、读专业文档的职场人，还是翻阅学术论文的研究者，ELA 让你专注阅读，无需切换标签或 App。


## 主要功能
- 文字转语音：由 OpenAI TTS 驱动，用自然流畅的声音朗读任意选中文字。
- 预置 AI 按钮：内置翻译和查词按钮，开箱即用。
- 完全可自定义：替换或扩展默认按钮，用自己的 prompt 定义功能，适配任何语言、学科或工作场景。
- 多语言支持：不限于英语。用 ELA 收听和学习任何语言的内容——德语、法语、日语等均可。自定义按钮可配置为任意语言对或学习任务。
- 灵活的声音设置：在设置中选择你偏好的 OpenAI TTS 声音模型。


## 如何配置
1. 打开扩展的"设置"页面。
2. 填入你的 OpenAI API Key。
3. 选择一个声音模型。
4. 可选：添加自定义 AI 按钮，填入你自己的 prompt。


## 如何使用
1. 打开侧边栏，开启右上角的开关。
2. 在网页上选中任意文字——内容将自动出现在侧边栏中。
3. 点击【播放】朗读文字。
4. 使用预置的【英翻中】和【查单词】按钮，或使用你在设置中配置的自定义按钮。


## 按钮说明
- 开/关：开启或关闭网页文字自动捕获。
- 设置：打开配置页面，设置 API Key、声音模型和自定义按钮。
- 清除：清空侧边栏中的所有内容。
- 播放：播放 / 继续播放 / 重新播放。
- 暂停：暂停播放。
- 停止：停止播放。下次点击播放时将重新生成音频。
- 英翻中 / 查单词：预置默认按钮，开箱即用。可在设置中修改或替换。
- 下载：将生成的音频下载为 MP3 文件。仅在音频成功生成后启用。
- 自定义按钮：在设置中定义你自己的 AI 功能，支持任意语言和任务场景。


## 隐私与权限
• 所有设置——包括你的 API Key 和自定义 prompt——均存储在本地 Chrome Storage 中，不会发送到 ELA 的任何服务器。
• 你的 API Key 仅用于向你配置的 AI 服务商（如 OpenAI）发起请求，开发者无法访问。
• 扩展仅在你主动操作时才会运行，不会在后台运行或监控你的浏览行为。
• storage 权限：将你的 API Key、声音模型偏好和自定义按钮 prompt 保存在本地设备上。

ELA 完全开源，你可以随时在 GitHub 上查看代码：https://github.com/janewu77/ela-extension
```

## 权限说明

_（CWS 开发者后台 → Privacy practices → Permission justification）_

**storage 权限说明**
用于在本地持久化保存用户设置，包括 OpenAI API Key、所选声音模型和自定义按钮配置。

**sidePanel 权限说明**
用于显示侧边栏界面，展示选中的文字、播放控件、翻译结果和查词结果。

**Host 权限说明**
用于捕获用户在任意网页上选中的文字，并发送到侧边栏进行处理（文字转语音、翻译等）。
